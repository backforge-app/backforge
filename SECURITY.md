# Security Policy for Backforge

## Supported Versions

At this stage of development, we only provide security updates for the latest branch/release of the project. 

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

Security is a top priority for us. If you discover a security vulnerability within this project, please **DO NOT** open a public issue. 

Instead, please send an email privately to: **aliskhandev@gmail.com**.

### What to include in your report:
* A description of the vulnerability and its impact.
* Steps to reproduce the issue.
* Any related code snippets or logs.

### Response Time
We will acknowledge receipt of your vulnerability report within 48 hours and strive to send you regular updates about our progress in fixing the issue. Once the issue is resolved, we will publish a patch and mention your contribution (if you prefer to be credited).
